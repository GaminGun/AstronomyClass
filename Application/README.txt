Для запуска программый на вашем компьютере должны быть установлены:
•	SQL Server;
•	SQL Server Management Studio.
Как только программы будут на вашем компьютере, запустите SQL Server Management Studio. При запуске вас попросят подключиться к серверу, выберите его тип: Ядро СУБД, затем напишите имя сервера: localhost\SQLEXPRESS. Нажмите кнопку «Соединить».

После соединения с локальным сервером откроется рабочее пространство и Обозреватель объектов. Если поле localhost\SQLEXPRESS свёрнуто, то разверните, в нём появятся рабочие объекты. Выберите объект «Базы данных» и нажмите на него правой кнопкой мыши, в контекстном меню выберите «Создать базу данных».

В открывшемся окне вы увидите поле «Имя базы данных». Впишите в него AstronomyClassDB и кликните по кнопке «ОК». После создания базы данных откройте папку с приложением и возьмите файл SCRIPT AstronomyClass.sql из папки Script в директории приложения, а затем перетащите его в рабочее пространство SQL Server Management Studio (оно тёмно-синее).

После загрузки скрипта рабочая область заполнится SQL-запросом. Вам необходимо нажать кнопку «Выполнить» над рабочей областью, либо нажать на рабочую область правой кнопкой мыши и нажать кнопку «Выполнить» в контекстном меню.

При нажатии кнопки «Выполнить» внизу рабочей области откроется поле терминала. Если вы увидите подобный вывод, то значит, что база данных успешно создана. Теперь можно переходить к запуску приложения.

После установки требуемых программ, настройки, а также копирования базы данных, перейдите в директорию скачанной программы «Класс Астрономии» и запустите файл AstronomyClassTest.exe.
